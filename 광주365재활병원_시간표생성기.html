광주365재활병원 주말 및 공휴일 시간표 생성기

사용 방법

  1. 시놀로지에서 신경계일일업무보고 시트를 엽니다.
  2. Ctrl + A 를 눌러 시트 전체 선택 후 Ctrl + C 를 눌러 복사합니다.
  3. 아래 큰 표의 아무 칸이나 클릭한 뒤 Ctrl + V 로 붙여넣습니다.
  4. 월 / 일 / 요일 입력 후 [현재 표를 주말 시트에 추가] 버튼을 누릅니다.
  5. 토요일 / 일요일 시트가 따로 있으면 각각 같은 방식으로 추가합니다.
  6. 환자 이름을 입력한 뒤 [개인 시간표 보기] 버튼을 누릅니다.

아직 추가한 시트가 없습니다.
